# MPD Backend — Michi Music Player

## Overview

MPD is integrated as a Hi-Fi backend for bit-perfect, DSD/DoP, and server-mode
playback. It is optional — GStreamer remains the default engine.

## Client (`audio/mpd/mpd_client.py`)

A minimal MPD client over TCP socket with no external dependencies.

### Supported commands

`ping`, `status`, `currentsong`, `play`, `pause`, `stop`, `seekcur`, `clear`,
`add`, `delete`, `next`, `previous`, `playlistinfo`, `outputs`,
`enableoutput`, `disableoutput`, `setvol`, `repeat`, `random`

### Protocol parser (`audio/mpd/mpd_protocol.py`)

Handles OK, ACK, key:value pairs, multiline list responses.

## Path Mapping (`audio/mpd/mpd_path_mapper.py`)

Michi uses absolute paths; MPD uses relative paths from its `music_directory`.
The mapper converts between the two:

```python
mapper = MpdPathMapper(music_dir="/home/user/Music")
mapper.to_mpd_path("/home/user/Music/Album/track.flac")  # → "Album/track.flac"
mapper.from_mpd_path("Album/track.flac")                  # → "/home/user/Music/Album/track.flac"
```

## Local MPD Service (`audio/mpd/mpd_service_manager.py`)

Michi can start/stop an isolated MPD instance without requiring root or
modifying `/etc/mpd.conf`.

### Configuration

Generated by `MpdConfigBuilder` and written to:
`~/.local/share/michi-music-player/mpd/mpd.conf`

Key settings:
- `bind_to_address`: `127.0.0.1` (local only)
- `port`: `6600` (configurable)
- `mixer_type`: `none` (for bit-perfect)
- `auto_update`: `yes` (watches music directory)

### Service management

```python
mgr = MpdServiceManager()
mgr.start(config)   # launches mpd with custom config
mgr.stop()          # sends SIGTERM
mgr.is_installed()  # checks if mpd binary exists
```

## MpdBackend (`audio/backends/mpd_backend.py`)

Implements the common `AudioBackend` API for MPD. Can connect to local or
remote MPD instances.

### Capabilities

- `supports_bitperfect`: True
- `supports_dsd`: True
- `supports_dop`: True
- `supports_remote`: True
- `supports_server_mode`: True
- `supports_digital_volume`: False

## Settings

| Key | Default | Description |
|-----|---------|-------------|
| `audio/mpd/enabled` | False | Enable MPD support |
| `audio/mpd/mode` | local | local / remote |
| `audio/mpd/host` | 127.0.0.1 | MPD host |
| `audio/mpd/port` | 6600 | MPD port |
| `audio/mpd/password` | "" | MPD password |
| `audio/mpd/music_directory` | ~/Música | Music directory for MPD |
| `audio/mpd/local_music_root` | ~/Música | Local path root for mapping |
| `audio/mpd/remote_music_root` | "" | Remote path root for mapping |
| `audio/mpd/auto_start_local` | True | Auto-start local MPD |
| `audio/mpd/bitperfect_verify` | True | Enable bit-perfect verification |
| `audio/mpd/dop_enabled` | False | Enable DoP for DSD |
